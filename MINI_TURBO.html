ÖPPNINGSBUD

1KL = 16+hp.

1RU = 2+ru, 13+:
      a) 13-15 bal el marmic (dock ej singel-ru).
      b) 13-15. Minst 4ru och 5+hö.
      c) 13-15. Minst 5-4 el 4-5 i lå.
    
1HÖ = Äkta 4+:
      a) 10-13 bal med 4hö eller 5hö+332.
      b) 4hö och 5+lå, 10-15.
      c) Enfärgad 6+hö, 12-15.
      d) Minst 5-4 i hö, 10-15.
         Vi öppnar med 4kortsfärgen, canapé.
      e) 13-15 marmic med singel-ru.

1NT = 10-13, ingen 4hö. Bal el obal.
      Minst 4kl eller 5ru, men ej 4333.

2KL = Minst 4kl och 5+hö. 13-15.

2RU = "Multi". a) 6hö eller tät 5hö. 8-11.
               b) 5hö+5ru, 13-15.

2HÖ = Minst 5hö + 4lå, 10-13. 
      Kan även vara 6hö+4lå eller 5-5.

2NT = 13-15. Enfärgad 6+lå.


Principer:
  Vi har ett överlapp ovan kring 13hp:
    13hp med Jxx, Qx, K, KJ, QJ, etc värderas som 10-13.
      Även 4333 kan räknas ner 1hp, 
      och mellankort (10'or, 9'or) kan förstås väga in.
    13-15 innehåller alltså från 13 'riktiga' hp.

  Med 13-15 5hö+5lå öppnar vi med lågfärgen (1RU el 2KL), 
    så att 5korts hö kommer fram i nästa bud.

Principer:

1. Generellt gäller efter alla sådana här transfers med 4+hö,
   om ÖH bjuder transferfärgen på 1-läget, så visar det exakt 3stöd.
   Med 4-stöd bjuder ÖH alltid färgen på 2-läget.
Exempel:
  1RU-(X)-1HJ[=4+sp]-1SP = Exakt 3sp.
                    -2SP = 4+sp.

2. Om mellanhanden bjuder icke-hoppande bud över SHs transfer-bud till 4+hö,
   så är ÖHs Dubbelt = support-dubbling, 3-stöd.
Exempel:
  1RU-(1HJ)-X[=4+sp]-(2KL)-X = Support-dubbling, 3stöd i sp.

3. Om SH bjuder ovanstående transfer-bud och ÖH återbjuder på 1-läget,
   så är 2LÅ i nästa rond alltid XY, förutom om SH upprepar
   överföring till transfer-färgen, varvid han visar 5+färg.

Exempel:
  1RU-(X)-1HJ[=4+sp]-1NT-2LÅ = XY.
                        -2HJ = Överföring till 5+sp.

  1RU-(X)-XX[=4+hj]-1SP-2KL = XY.
                            -2RU = INTE XY! Överföring till 5+hj.

ÖPPNINGSBUDET 1NT

Visar 10-13, ingen 4hö. Bal el obal. Minst 4kl eller 5ru, men ej 4333.

1NT-2NT = Transfer till 3KL.
1NT-3KL = Passa med 4+kl, annars bjud 3RU(5+ru).
1NT-3NY = Spärrbud.

1NT-2KL = Krav med utgångsintresse.
       -2RU = Ingen singel.
           -2HJ = UK. Frågar efter håll i hö.  Svar ???
           -3HÖ = UK, fastställer trumf.
           -Övriga = Invitstyrka.
       -2HJ = 1-3 i hö.
       -2SP = 3-1 i hö.
       -2NT = 5-5 i lå.
       -3KL = 4-6 i lå.
       -3RU = 6-4 i lå.
       -3HJ = 3-3-1-6.
       -3SP = 3-3-6-1.

1NT-2HÖ = Exakt 5korts (ej tät). Ej utg-intresse. Med singel föreslår ÖH normalt nåt annat.

1NT-2RU = 6korts hö, eller tät 5korts.
       -2HJ = Förnekar hj-anpassning, dvs ej 3korts eller Hx i hj.
       -2SP = Hj-anpassning, men ej sp-anpassning.
       -2NT = Anpassning i båda hö, min.
       -3XX = Anpassning i båda hö, max.
       
Störd 1NT:

1NT-(X)-XX  = Transfer till 5+klöver eller 5+ruter.
       -2KL = Ber ÖH välja lågfärg.
       -2RÖ = Transfer till 5+ i nästa färg.

1NT-(2HÖ)-2NT = Lebensohl, normalt avlägg i lägre färg.
1NT-(2HÖ)-3KL = Lågfärgsväljare (normalt minst 3ru+4kl).

ÖPPNINGSBUDET 2KL

2KL visar minst 4kl och 5+hö, 13-15.

2KL-2RU = Utgångs-intresse med allmän styrka eller 5+hjärter.
        -2HJ = 5+sp. xx eller  sämre i hj.  SH kan passa med lång hj, eller -2SP=sign-off.
        -2SP = 5+sp. Minst Hx el xxx i hj.  SH kan passa eller korrigera till hj.
        -Högre bud = 5+hjärter. En enkel, naturlig modell:
        -2NT = 5hj+4kl.
        -3KL = 5hj+5kl.
        -3HJ = 6hj+4kl.

2KL-2HJ = Primärt Pass-or-correct.
          Men kan ha utgång-intresse mot 5+spader hos ÖH.
2KL-2HJ-2SP-2NT = Krav med god sp-anpassning.
2KL-2HJ-2SP-3HJ = Utgångsinvit med hjärter spelbar till renons.
                  ÖH accepterar med bra topphonnörer.

2KL-2SP = Okrav med egen färg, och kort hj.
          Öppnaren höjer med god anpassning.

ÖPPNINGSBUDET 2RU

2RU visar 
a) 8-11, och 6korts hö eller tät 5korts hö (typ AQJTx eller liknande).
b) 5hö+5ru, 13-15.

2RU-2HÖ = Normala multi-svar. Pass-or-correct.
2RU-3KL = Utg-krav med egen 5+hj. Max 2sp.
2RU-3RU = Utg-krav med egen 5+sp. Max 2hj.
2RU-3HJ = Spärrhöjn till båda hö, ej invit.
2RU-3SP = Utg-krav i öppnarens hö.

2RU-2HÖ-PASS/2SP = 5+hö, 8-11.
       -2NT = 5hj+5ru, 13-15.
       -3KL = 5sp+5ru, 13-15.

2RU-2NT = Krav och minst utg-invit oavsett vilken hö öppnaren har.
        -3KL = 6korts hö.
        -3RU = Tät 5korts hj.  -3HJ = Inviterande okrav.
        -3HJ = Tät 5korts sp.  -3SP = Inviterande okrav.
        -3SP = 5hj+5ru, 13-15.
        -3NT = 5sp+5ru, 13-15.

2RU-2NT-3KL = Någon 6korts hö.
           -3RU = Kräver till utgång, och öppnaren bjuder högfärg KORSVIS!
           -3HJ = Utg-invit till hj-hand. Och minst utg-invit till spader-hand.
                    SH passar eller bjuder ut sin hand.
           -3SP = Utg-krav till hj-hand, invit till sp-hand.

Störd 2RU:

Om motståndarna på 2RU-Multi kliver in med 2/3/4 i HÖ, gäller: 
  X = Ej straffdubbling!
      ÖH passar endast med 5kort i färgen, annars bjud!
Anledningen är att SH med korthet i inklivsfärgen 
  och längd i objudna hö annars hamnar i omöjligt läge.
Budgivningen kan t ex gå: 2RU-(4SP), och SH har 0sp+5hj. 
Nu kan han Dubbla, partnern kommer endast passa med 5+spader.
  
2RU-(X)-PASS = kort hj, ruter-längd.
        2HJ  = Pass-or-correct.

ÖPPNINGSBUDEN 2HÖ

Visar minst 5+hö och 4+lå, 10-12.

2HJ-2SP = Okrav med egen färg.

2HÖ-3KL = Pass-or-correct, dvs avlägg i öppnarens lågfärg.

2HÖ-3RU = Utgångs invit med 3+stöd i högfärgen.

2HÖ-3HÖ = Spärrhöjning, ej invit.

2HÖ-2NT = Kravbud. Utg-krav eller avlägg med egen lågfärg.
          Detta ber ÖH bjuda sin KORTA lågfärg.
          Detta för att SH ska kunna stanna i egen lång lågfärg.
          ÖH kan dock svara högre med tät 6korts hö o spelstark hand.
          Om SH återkommer är det utg-krav.
