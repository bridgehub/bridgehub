ÖPPNING 1KL

Positiv SH: är EE eller 2c+3g (EQQ, KKQ) eller bättre.

Styrkeskalan för positiva händer efter relä:
   - EE eller 2c.
   - 6n3g.
   - 7n3g.
   - 7n4g.
   - 8n3g.
   - 8n4g.
   - 8n5g.
   - 9n3g.
   - 9n4g.
   - 9n5g.

Div regler i 1KL-budgivning:

- Om ÖH under 3NT *inte* bjuder reläbudet, är det naturligt.
    Fastställer trumfen om ÖHs längd i färgen är känd.

- Positiv svarshand med minst 4-4 i 2 färger, måste ha minst 2G i de långa färgerna.

- Om SHs styrkeskala startar i 3NT slås de 2 första svaren ihop i första steget.

- Om ÖH reläar förbi 3NT och SH visat hö+lå:
  -4KL relä fastställer högfärgen.
  -4RU relä fastställer lågfärgen.

- 1KL-2bud: Med bara 2c, krävs G i sidofärgen, annars 1KL-1RU.


  1KL-1RU = a) Negativ.
            b) Positiv, minst 4-4 i hö.
            c) Positiv, GGxxx+332 i övriga.

Övriga svar positiva:

  1KL-1HJ = Ej hj, dvs 4+sp eller ingen 4hö.
     -1SP = 4+hj, ej 4+sp.
     -1NT = 6+hö.
     -2KL = GGxxx(x) i ngn lå + sidofärg.
     -2RU = GGxxx i hj + sidofärg.
     -2HJ = GGxxx i sp + lågfärg.
     -2SP = GGxxxx i lå, ej 4hö.
     -2NT = GGxxx i sp. 4hj el 5-5.


1KL-1RU

  1KL-1RU-1HJ = Krav. a) 4+hj obal. Ej minimum med 4hj+5kl eller 5hj+4sp.  
                                              ==> -1SP-2KL/2RU/2HJ

                      d) 22+NT, ej 4sp.       ==> utan hj: -1SP-2NT/3NT
                                              ==> med 4hj: -1SP-2KL-2RU-2NT/3NT
                      e) UK utan sp, förutom 5hj+4sp. 
         -1SP = Krav med 4+sp, obal. Ej minimum med 6sp eller 5kl+4sp.
         -1NT = 16-17 bal.
         -2KL = 6kl eller 5kl+4hö.  -2RU = Högfärgsfråga.
         -2RU = a) 20-21 bal.  
                b) 24-25 bal utan 4hö.  
                c) 4-1-4-4 17-24hp.
         -2HJ = 5+hj, 4sp. Minimum.
         -2SP = 6sp. Minimum.
         -2NT = 22-23 bal. 4sp.
         -3LÅ = 6lå, ej sidofärg. 4F, ca 19-21hp.
         -3HÖ = Tät 6hö, 9ss. Spelbar till singelton.
         -3NT = 24-25 bal. 4sp.

             b) (Semi)solid 6+kl, ca 7 spelstick.
             c) 13-15. 7-korts klöver.

        -2HJ = 5+sp. xx eller sämre i hj.
        -2SP = 5+sp. Minst Hx el xxx i hj.
        -2NT = a) 6+klöver-hand. b) 6hö+5kl. c) 6ru+4kl.
              -3KL = Avlägg till kl-handen (SH har 5+hj).
                     ÖH bjuder vidare nat med 6ru eller 6hö+5kl.
              -3RU = Krav.
                  -3HÖ  = 6hö+5kl.
                  -3NT  = (Semi)solid 6kl, ca 7 spelstick.
                  -4KL  = 7+kl.     -4NT = Slutbud.
                  -4RU  = 6ru+4kl.  -4NT = Slutbud.
        -3KL = 2-5-2-4. 
             -3RU-3HJ+ = Ingen Cx, Cx i lägre, Cx i båda.  (C=E/K)
        -3RU = 1-5-3-4.
        -3HJ = 2-5-2-4, Cx i spader.
        -3SP = 3-5-1-4.
        -3NT = x-5-x-5.

2KL-2HJ = Primärt avlägg i ÖHs förmodade högfärg. 
          Kan ha utgångs(slam)-intresse mot 5+spader hos ÖH.

ÖPPNING 2RU MULTI

  a) 8-11, och 6korts hö eller tät 5korts hö (typ AQJTx eller liknande).
  b) Spelstark 5hö+5ru, 14-15.

2RU-3HJ = Spärrhöjn till båda hö, ej invit.

2RU-2HÖ-PASS/2SP = 5+hö, 8-11.
       -2NT = 5hj+5ru, spelstark 14-15.
       -3KL = 5sp+5ru, spelstark 14-15.

        -3KL = 6korts hö.
        -3RU = Tät 5korts hj.  -3HJ = Inviterande okrav.
        -3HJ = Tät 5korts sp.  -3SP = Inviterande okrav.
        -3SP = 5hj+5ru, spelstark 14-15.
        -3NT = 5sp+5ru, spelstark 14-15.

           -3RU = Kräver till utgång, och öppnaren bjuder högfärg KORSVIS!
           -3HJ = Utg-invit till hj-hand. Och minst utg-invit till spader-hand.
