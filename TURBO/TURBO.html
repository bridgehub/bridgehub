ÖPPNING 1KL

Positiv SH: är EE eller 2c+3g (EQQ, KKQ) eller bättre.

Styrkeskalan för positiva händer efter relä:
   - EE eller 2c.
   - 6n3g.
   - 7n3g.
   - 7n4g.
   - 8n3g.
   - 8n4g.
   - 8n5g.
   - 9n3g.
   - 9n4g.
   - 9n5g.

  1KL-1RU = a) Negativ.
            b) Positiv, minst 4-4 i hö.
            c) Positiv, ggxxx+332 i övriga.
          
Övriga svar positiva:

  1KL-1HJ = Ej hj, dvs 4+sp eller ingen 4hö.
     -1SP = 4+hj, ej 4+sp.
     -1NT = 6+hö.
     -2XX = Minst ggxxxx i lå eller någon ggxxx+sidofärg.

  1KL-1HJ-1SP = Relä med  4+sp.  SH forts -1NT med 4+sp, eller -2KL+ utan hö.
         -1NT = Relä utan 4+sp.  SH forts -2KL utan hö,  eller -2RU+ med 4sp.

Exempel på reläskalan då SH har 4hö -- här 4+hj:

  1KL-1SP-1NT-2KL  = 4hj, 4+kl (inkl 1-4-4-4).
                     -2RU = Relä utan 4hj.
                     -2HJ = Relä med  4+hj. För att få ev hj-kontrakt på rätt hand
             -2RU  = 5hj.
          !! -2HJ    ANVÄNDS EJ PGA ATT HJ-KONTRAKT KAN KOMMA PÅ FEL HAND!
             -2SP  = 2-4-4-3 eller x-4-6-x.
             -2NT  = 3-4-4-2.
             -3KL  = 3-4-3-3 eller 0-4-5-4.
                    (i kl-fallet: 1-4-4-4 eller 0-4-4-5)
             -3RU  = 2-4-5-2.
             -3HJ  = 1-4-5-3.
             -3SP+ = 3-4-5-1. Styrke-skalan...
             
I spaderfallet:

  1KL-1HJ-1SP-1NT(4+sp)-2KL => samma skala som ovan, dvs:
                           -2RU = 4sp, 4+kl (inkl 4-1-4-4).
                           -2HJ = 5sp.
                           -2SP = 4-2-4-3 eller 4-x-6-x.
                           -2NT = 4-3-4-2.
                           -3KL = 4-3-3-3 eller 4-0-5-4.
                                 (i kl-fallet: 4-1-4-4 eller 4-0-4-5)
                           -3RU = 4-2-5-2.
                           ... etc ...
alternativt:

  1KL-1HJ-1NT-2KL  = Ingen hö.
             -2RU+ = 4+sp. Samma skala som ovan.

Visar minst 5+hö och 4+lå, 10-12.

2HJ-2SP = Okrav med egen färg.

2HÖ-3KL = Pass-or-correct, dvs avlägg i öppnarens lågfärg.

          Om SH återkommer är det utg-krav.
