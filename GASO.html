GA-SO  ---  Mini-Turbo



ÖPPNINGSBUD

1KL = 16+. 

1RU = 2+ru:
      a) 10-12 bal utan hö.
      b) 6+ru, 10-12.
      c) 4ru och 5+hö, 13-15.
      d) 54/45 i lå, 10-15.
    
1HÖ = Äkta 4+:
      a) 10-12 bal med 4hö eller 5hö+332.
      b) 4hö och 5+lå, 10-15.
      c) Enfärgad 6+hö, 12-15.
      d) Minst 5-4 i hö, 10-15. 
         Vi öppnar med 4kortsfärgen, canapé.

1NT = 13-15 bal. Stayman, transfers.

2KL = Minst 4kl och 5+hö. 13-15.

2RU = "Multi". 6hö eller tät 5hö. 8-11.

2HÖ = Minst 5hö + 4lå, 10-12. Kan även vara 6hö+4lå eller 5-5.

2NT = 13-15. 6+lå.


Princip:
Med 13-15 5hö+5lå öppnar vi med lågfärgen (1RU el 2KL), 
så att 5korts hö kommer fram i nästa bud.

ÖPPNINGSBUDET 1RU

Över 1RU bjuder SH lägsta hö med minst 3kort 
  (öppnaren har aldrig 4hö, men kan ha 5+hö), alltså:
 1RU-1HJ = Minst 3hj. OBS! kan innehålla längre spader än hjärter!
 1RU-1SP = Max 2hj, minst 3sp.
 1RU-1NT = Max 2-2 i hö.

Efter 1RU-1HÖ-1NT använder SH XY-NT med minst utg-invit.
  2HÖ från ÖH visar sedan 3korts, medan 2HÖ från SH visar 5+färg.
  OBS! Att 1RU-1HJ kan innehålla längre spader än hjärter(3+)!

OBS! Att 1RU-1HJ-1NT-2SP visar avläggshand med minst 3hj och 5+sp (ej "reverse"!).

Öppning 1RU och återbud 2RU på nästa varv är begränsat 10-12, eftersom vi öppnar 2NT med 13-15.

Öppning 1RU och återbud 2KL på nästa varv visar 13-15, 
men kan vara antingen minst 5-4 eller 4-5 i lå.

Efter 1RU-2KL(konv krav, minst invit till 10-12) 
bjuder ÖH alltid 2RU med 10-12 oavsett fördelning (bal, 6+ru eller 54/45 i lå).
  1RU-2KL-2RU = 10-12 oavsett fördelning (men förnekar 4ko hö).
  1RU-2KL-2HÖ = 5+hö, 13-15.
  1RU-2KL-2NT = Minst 4ru+5kl, 13-15   ("transfer" till 5+lå).
  1RU-2KL-3KL = Minst 5ru+4kl, 13-15   ("transfer" till 5+lå).



ÖPPNINGSBUDET 1HÖ

Vi bjuder canapé även med 5-4 (6-4) i hö och öppnar med 4kortsfärgen.

1HÖ-2NT = Stenbergs, invit+. 
          SKROT gäller med undantaget att 
          1HÖ-2NT-3KL-3HÖ är okrav med invithanden (ej singelvisning).

1SP-1NT-2HJ = Nat 10-12.

1HÖ-1NT-2RU = 13-15 4hö och 5+ruter.

1HÖ-1SP/1NT-2NT = 13-15 4hö och 5+klöver, varpå -3KL avlägg.

1HÖ-1SP/1NT-2HÖ = 6hö 12-13.

1HÖ-1SP/1NT-2KL = Konv. Antingen 6hö 14-15, eller 5-4 i hö 13-15.  -2RU relä.

1SP-2HJ = Exakt 5hj, inget utgångsintresse. ÖH kan passa, eller med singel-hj föreslå 2SP, 2NT eller 3LÅ(canapé) som slutbud.

1SP-2RU = a) 6hj. Avlägg, invit el UK.              Återbud: 1SP-2RU-2HJ-3XX = 6hj, invit+.
          b) Exakt 5hj. Stark invit, ca 12-13(14).  Återbud: 1SP-2RU-2HJ-2NT.
          c) Exakt 5hj. Utg-krav.                   Återbud: 1SP-2RU-2HJ-2SP*.

1SP-2KL = Artificiellt, minst utgångsinvit mot 10-12.
       -2RU = Minimum, men ej canapé-hjärter.
           -2HJ = Exakt 5hj. Mild invit, ca (9)10-11.
           -2NT = Naturlig invit till 10-12-hand.
                  ÖH kan föreslå 3LÅ som slutbud med lämplig (6+?)canapé, 10-11.
       -2HJ = Minst 5+hj. Canapé. Okänd styrka.
       -2SP = 6+sp, 12+p. UK.
       -2NT = 5+kl, 13+. UK.
       -3KL = 5+ru, 13+. UK.

ÖPPNINGSBUDET 2KL

2KL visar minst 4kl och 5+hö, 13-15.

2KL-2RU = Frågar efter högfärgen, lovar ej invitstyrka.
          SH kan sen passa eller höja 2HÖ till 3HÖ inviterande.
          Även 2KL-2RU-2HÖ-2NT spelas nog enklast som inviterande.

2KL-2HÖ = Okrav med egen färg. Öppnaren höjer med god anpassning.



ÖPPNINGSBUDET 2RU

2RU visar 8-11, och 6korts hö eller tät 5korts hö (typ AQJTx eller liknande).

2RU-2HÖ = Normala multi-svar. Pass-or-correct.
2RU-3KL = Utg-krav med egen 5+hj. Max 2sp.
2RU-3RU = Utg-krav med egen 5+sp. Max 2hj.
2RU-3HJ = Spärrhöjn till båda hö, ej invit.
2RU-3SP = Utg-krav i öppnarens hö.

2RU-2NT = Krav och minst utg-invit oavsett vilken hö öppnaren har.
        -3KL = 6korts hö.
        -3RU = Tät 5korts hj.  -3HJ = Inviterande okrav.
        -3HJ = Tät 5korts sp.  -3SP = Inviterande okrav.

2RU-2NT-3KL = Någon 6korts hö.
           -3RU = Kräver till utgång, och öppnaren bjuder högfärg KORSVIS!
           -3HJ = Utg-invit till hj-hand. Och minst utg-invit till spader-hand.
                    SH passar eller bjuder ut sin hand.
           -3SP = Utg-krav till hj-hand, invit till sp-hand.

Störd 2RU:

2RU-(X)-PASS = kort hj, ruter-längd.
        2HJ  = Pass-or-correct.



ÖPPNINGSBUDET 2HÖ

Visar minst 5+hö och 4+lå, 10-12.

2HJ-2SP = Okrav med egen färg.

2HÖ-3KL = Pass-or-correct, dvs avlägg i öppnarens lågfärg.

2HÖ-3RU = Utgångs invit med 3+stöd i högfärgen.

2HÖ-3HÖ = Spärrhöjning, ej invit.

2HÖ-2NT = Kravbud. Utg-krav eller avlägg med egen lågfärg.
          Detta ber ÖH bjuda sin KORTA lågfärg.
          Detta för att SH ska kunna stanna i egen lång lågfärg.
          ÖH kan dock svara högre med tät 6korts hö o spelstark hand.
          Om SH återkommer är det utg-krav.



ÖPPNINGSBUDET 2NT

Öppning 2NT visar någon 6korts lå, 13-15.

2NT-3KL = Pass-or-correct.

2NT-3HÖ = Okrav med 6+hö.

2NT-3RU = Söker stöd (xxx el Hx) till 5+hö.  
       -3HJ = Förnekar hj-stöd. 
       -3SP = Visar hj-stöd, men förnekar sp-stöd.
       -3NT = Stöd i båda.
